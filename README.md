# Hangman Game

This is a simple Hangman game implemented with Node.js, Express, PostgreSQL, and Redis. The application is containerized using Docker and can be deployed to Kubernetes.

## Prerequisites

- Docker
- Docker Compose
- Kubernetes
- Helm
